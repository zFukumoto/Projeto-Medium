# Projeto Medium

Este projeto é um clone da interface do site Medium, desenvolvido para fins educacionais.

## Objetivo

Praticar HTML e CSS recriando a estrutura e o layout de um site real.

## Tecnologias utilizadas

- HTML5
- CSS3

## Funcionalidades

- Header
- Topo com Botão
- Área de Conteúdos
- Footer

## Observação

Este projeto foi desenvolvido apenas para estudo e prática de desenvolvimento front-end.
Não possui qualquer vínculo com o Medium oficial.
